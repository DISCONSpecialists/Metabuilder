March 03, 2008 1.6.2984.25794 Full Installer
  • FIXED: Printing - Number of Copies now working correctly
  • FIXED: Artefact Text Boxes resizing incorrectly when editing
    NOTES:
    Existing diagrams with artefacts need to be saved and reopened for this change to take effect
  • FIXED: Globalization / Regional changes differing from machine to machine might cause the following: NOTES:
    User A creates a diagram using en-ZA encoding, and saves the file.
    User B opens the diagram, with only en-US encoding installed.
    Since the datetime formats of these machines differ, User B cannot open the file.

    The code now iterates through all installed cultures when it finds that it cannot parse any datetime.
    If it still cannot parse, Today's date is returned.
  • FIXED: Certain icons in old diagrams [SystemComponent] loses transparency when opened with the new update
